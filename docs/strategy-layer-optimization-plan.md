# 《万历：山河崩塌》策略层深度优化计划

> 参考游戏：Victoria 3（利益集团/法律/外交博弈/市场/建设队列/人口分层）
> 设计目标：在"低操作量·高战略深度"的设计原则下，将当前 MVP 的单层决策扩展为多维策略博弈
> 编写日期：2026-06-29

---

## 一、当前状态诊断

### 1.1 已有策略层
| 决策维度 | 当前形式 | 深度评价 |
|---|---|---|
| 军事方向 | 1 个下拉框（选择攻击目标） | 浅——无战争目标、无和谈条件、无多线作战压力 |
| 内政重点 | 6 项单选（农桑/财政/军备/吏治/休养/边疆） | 浅——无代价权衡、无制度切换、无长期路径选择 |
| 军事姿态 | 3 档（守势/均衡/攻势） | 中等——已影响战斗计算，但无战略代价 |

### 1.2 缺失的策略维度
- **外交**：不存在。AI 势力之间没有结盟、朝贡、互市。
- **内部分化**：不存在。朝堂是"一个人"的意志，无派系博弈。
- **制度选择**：不存在。税制、兵制、科举制度是"默认值"，不可改变。
- **区域建设**：不存在。不能修建水利/城防/卫所。
- **区域贸易**：不存在。南粮北运、边贸互市不影响经济。
- **战争目标**：不存在。战争只有"进攻"和"占领"，无有限战争/和谈/册封。

## 二、维多利亚3核心设计模式 → 万历化映射

V3 设计的核心思想是："国家不是一个人，而是利益博弈的容器。玩家不是独裁者，而是调停者。" 这个思想可以完美适配晚明的叙事。

| Victoria 3 系统 | 万历化映射 | 设计价值 |
|---|---|---|
| **利益集团** (Interest Groups) | **朝堂派系**：东林党、阉党、楚党、浙党、勋贵、军功集团 | 内政决策不再是自由选择，而是需要争取派系支持 |
| **法律与制度** (Laws & Institutions) | **国策**：税制(实物/一条鞭/加派)、兵制(卫所/募兵/营兵)、科举(八股/实务)、海禁(开/闭) | 重大制度变更需要"朝堂博弈"，有拥护和反对的派系 |
| **外交博弈** (Diplomatic Plays) | **朝贡与边策**：对蒙古(互市/和亲/征伐)、对女真(羁縻/册封/征伐)、对日本(海禁/勘合贸易) | 战争不再是点击即打，而是通过"边策等级"逐步升级 |
| **建筑队列** (Construction Queue) | **区域建设**：水利、城防、卫所、驿道、市舶司 | 建设消耗国库/粮食，产出人口承载力/防御/税收 |
| **市场与贸易** (Markets) | **粮道与贸易网**：区域间粮食调配、东南沿海贸易、西北茶马互市 | 贸易路线影响粮价、区域稳定和势力收入 |
| **人口分层** (Pops) | **阶层**：士绅、自耕农、佃农、工匠、商人、军户 | 不同阶层响应不同政策，叛乱由具体阶层驱动 |
| **期刊** (Journal Entries) | **事件链**：已有25个事件基础，可升级为有"完成条件"的事件链 | 给玩家中短期目标，完成有奖励，放弃有惩罚 |

## 三、分阶段实施计划

### 阶段 A：朝堂派系 + 内政代价（优先级：最高）

**核心改动**：将"6 项内政重点"的单选改为有代价的权衡系统。

#### A.1 朝堂派系系统
```
朝堂派系（3-5 个，动态增减）
├── 东林党      诉求：澄清吏治、减税、开海禁
├── 阉党/内廷   诉求：加强皇权、增加矿税、压制文官
├── 地方缙绅    诉求：维持地方特权、反对清丈田亩
├── 军功集团    诉求：增加军费、边功封赏、反对裁军
└── 西北边将    诉求：边疆自主、互市收益、反对调防
```

- 每个派系有 **支持度**（0-100）和 **力量权重**（受区域内人口阶层分布影响）。
- 内政重点的切换不再是自由选择，而是：**每个选择会升降各派系支持度**。
- 如果主导派系支持度过低，触发"掣肘"减益：行政效率下降、税收征集困难、或军队不服从。
- 派系支持度高时提供正面 buff。

#### A.2 内政决策的三层结构（取代原 6 项单选）
```
决策层 1：长期国策方向（每 2-3 年可改一次）
  ├── 重农抑商 → 农业产出+15%，商业税收-20%
  ├── 开海通商 → 沿海贸易收入+30%，沿海防御-10%
  ├── 强军固边 → 军队维护费+25%，军事组织+10
  ├── 整顿吏治 → 腐败-5/月，行政效率+8
  └── ...（6-8 项）
  
决策层 2：当前内政重点（每月可调，原 6 项保留但加上派系反应）
  
决策层 3：紧急措施（一次性消耗合法性/国库以应对危机）
  ├── 加派辽饷 → 国库+，全国稳定-
  ├── 开仓赈灾 → 粮食-，目标区域稳定+
  ├── 裁撤冗兵 → 军费-，军队规模-
  └── ...
```

**实施难度**：中。需要新增派系数据模型、支持度计算、UI 面板重构。

---

### 阶段 B：国策制度切换 + 朝堂博弈（优先级：高）

**核心改动**：将事件驱动的改革（如一条鞭法、清丈田亩）转为玩家可主动推动的制度变革。

#### B.1 制度面板
```
当前制度（类似 V3 的 Laws）
├── 税制    ：[实物税] → [一条鞭法] → [加派赋税]
├── 兵制    ：[卫所制] → [募兵制] → [营兵制]
├── 科举    ：[八股取士] → [实务取士]
├── 海禁    ：[禁海] → [勘合贸易] → [开海通商]
├── 边疆    ：[羁縻卫所] → [改土归流]
└── 财政    ：[太仓库] → [内帑分离] → [统一财税]
```

- 切换制度需要 **朝堂支持**：启动后进入 N 回合的"推行期"，每回合有派系阻挠事件。
- 推行成功 → 切换为新制度（永久 buff + debuff）。
- 推行失败 → 浪费国库、合法性下降、可能触发某些派系的叛乱。

#### B.2 博弈机制
- 可用"朝廷影响力"点数（由合法性和行政效率决定）来推动或安抚派系。
- 特殊事件：如"皇上怠政"时期，朝堂博弈被冻结或由 AI 派系主导。

**实施难度**：高。需要制度数据模型、推行期逻辑、博弈 UI。

---

### 阶段 C：边策与外交（优先级：中）

**核心改动**：将当前的"选目标→开战"改为四层边策梯度。

#### C.1 边策梯度（代替当前纯军事目标）
```
对任意接壤势力：
├── L1：互市通商  → 双边贸易收入+，对方军事组织缓慢增长
├── L2：羁縻册封  → 对方合法性受控，我方需支付岁币/赐赏
├── L3：外交孤立  → 我方消耗外交影响力，削弱对方贸易与合法性
├── L4：军事征伐  → 进入战争状态（当前已实现的基础）
```

- 每级边策有月度消耗（影响力、国库、粮食）和效果。
- L1→L4 升级有冷却和条件（如对方侵略行为可跳级）。
- L1→L3 可以同时存在（对不同方向的不同势力采取不同策略）。

#### C.2 战争目标系统（替代"攻占即永久控制"）
```
战争目标类型：
├── 收复失地  → 攻占后控制权转移，战后永久归属
├── 边疆推进  → 攻占后控制权临时转移，战后部分保留
├── 惩戒征伐  → 不追求领土，降低对方军事/经济
├── 扶植政权  → 攻占后成立/转移给傀儡势力
└── 全面征服  → 要求对方投降/灭国
```

- 战争开始前宣告目标，达成后进入和谈。
- 和谈条件基于"战争分数"（控制了多少目标区域 + 消灭多少敌军）。

**实施难度**：高。需要外交数据模型、和谈逻辑、边策 UI。

---

### 阶段 D：区域建设队列（优先级：中）

**核心改动**：从一个静态的区域面板变为可以投资建设。

#### D.1 建设类型
```
区域建设项目：
├── 水利灌溉  → 消耗国库/粮，提高人口承载力 +15%，粮食产出 +10%
├── 城墙加固  → 消耗国库，提高城防值 +20%
├── 卫所屯田  → 消耗国库/粮，增加驻军上限 +2000，驻军粮食消耗 -10%
├── 驿道修整  → 消耗国库，提高控制度恢复速度 +1/月
├── 市舶司    → 消耗国库，沿海区域商业税收 +25%，开启贸易路线
└── 仓储备荒  → 消耗粮，提高存粮上限，饥荒时稳定度下降减半
```

#### D.2 机制
- 每个区域有一个建设队列，可排队 1 个项目。
- 建设消耗国库/粮食，需要 N 个月完成。
- 完成后提供永久 buff。
- 全国可同时建设的项目总数受制于"行政效率"。

**实施难度**：中。需要建设数据模型、队列 UI。

---

### 阶段 E：阶层人口与经济联动（优先级：低）

**核心改动**：在现有 population 基础上细分阶层，让人口结构影响派系力量和政策效果。

#### E.1 阶层分类
```
阶层      占比       主要区域        政治倾向     叛乱类型
士绅      5-8%      江浙/北直隶    保守/东林    朝堂政变（低概率）
自耕农    40-50%    中原/华北      均衡/重农    抗税暴动
佃农      15-25%    江南/湖广      激进/减租    佃变
工匠      3-5%      沿海/城市      开海/商业    罢市
商人      3-5%      沿海/运河      开海/低税    走私
军户      8-12%     边疆/卫所      强军/边功    哗变
```

- 不同政策（税制、兵制、海禁）影响不同阶层的满意度。
- 阶层满意度汇总为该区域的稳定度修正。
- 叛乱爆发时根据该区域主要阶层确定叛乱类型和叛军特征。

**实施难度**：高。需要阶层数据模型和全系统联动。

---

### 阶段 F：事件链 + 期刊目标（优先级：中）

**核心改动**：将单次弹出事件升级为有多个阶段的"事件链"，类似 Victoria 3 的 Journal Entries。

#### F.1 事件链结构
```
事件链：张居正改革
├── 阶段 1：考成法推行（已触发 → 触发下一阶段条件）
├── 阶段 2：清丈田亩（触发条件：考成法完成 + 日期窗口）
├── 阶段 3：一条鞭法推广（触发条件：清丈田亩完成）
├── 阶段 4：改革派系崛起（触发条件：三条均完成）
└── 阶段 5：张居正去世后的清算or延续（触发条件：1582年）

事件链：辽东危机
├── 阶段 1：建州整合（已触发）
├── 阶段 2：后金建立（条件：建州整合 + 1616年）
├── 阶段 3：抚顺失守（条件：后金建立 + 明辽防虚弱）
├── 阶段 4：萨尔浒（条件：抚顺失守）
└── 阶段 5：辽沈崩溃 or 熊廷弼防线（取决于玩家选择）
```

- 每个事件链有 **完成/失败条件**。
- 完成提供奖励 buff，失败导致 debuff 并可能触发新事件链（如"流民扩散"→"李自成崛起"）。
- 同时活跃的事件链上限为 3 个，超过则旧链封存。

**实施难度**：低-中。当前事件系统已有基础，需要扩展链式结构和条件。

## 四、实施优先级与依赖关系

```
阶段 A：朝堂派系 + 内政代价 ──────── 最高优先级
  │ 依赖：无
  │ 效果：立即让内政决策从"无脑单选"变成"权衡博弈"
  │
  ├─→ 阶段 B：国策制度切换 ─────── 高优先级
  │     依赖：阶段 A（派系是制度变更的阻力来源）
  │
  ├─→ 阶段 C：边策与外交 ───────── 中优先级
  │     依赖：阶段 A（派系影响外交取向）
  │
  ├─→ 阶段 D：区域建设队列 ─────── 中优先级
  │     依赖：无（可独立实施）
  │
  ├─→ 阶段 F：事件链 ──────────── 中优先级
  │     依赖：无（可独立实施）
  │
  └─→ 阶段 E：阶层人口 ─────────── 低优先级
        依赖：阶段 A（阶层决定派系力量）
```

## 五、MVP 后的推荐实施顺序

1. **立即（下次迭代）**：阶段 A 的核心部分——引入 3-5 个朝堂派系，内政重点加入派系反应。
2. **短期（1-2 周内）**：阶段 F 事件链——将现有 25 个事件串联为 5-6 条链。
3. **中期**：阶段 C 边策梯度——用四层边策替代纯军事目标。
4. **中期**：阶段 D 区域建设——增加战略投资的维度。
5. **长期**：阶段 B 国策制度——全面制度切换系统。
6. **长期**：阶段 E 阶层人口——最深度的系统联动。

## 六、设计原则重申（对照 V3）

| V3 原则 | 万历化应用 |
|---|---|
| "国家不是一个人" | 朝堂派系代表不同利益，玩家调停而非独裁 |
| "改变制度需要博弈" | 国策变更需要派系支持 + 多回合推行 |
| "外交是战争的延伸" | 边策梯度让战争成为"最后手段"而非"唯一手段" |
| "经济驱动政治" | 粮食/贸易/税收影响派系力量和区域稳定 |
| "人口不是数字" | 阶层分化让人口变化有政治含义 |
| "玩家设定方向，系统负责演算" | 保持"低操作量·高战略深度"核心设计 |

所有改动将严格遵循项目既有的六条设计原则（操作少·决策重·后台复杂前台简单·事件给压力不强制历史结果·地图控制权优先·统一规则不偏袒）。
